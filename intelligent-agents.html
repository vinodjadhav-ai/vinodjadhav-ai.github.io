Intelligent Agents

1. Module Overview

This module focuses on agent-based systems, autonomous decision-making, intelligent agent architectures, and how agents can interact with environments and other agents to solve real-world problems.

2. Learning Outcomes

  • Identify and critically analyse agent-based systems, differentiating between architectures and approaches.
  • Apply and critically evaluate intelligent agent techniques to real-world problems where uncertainty and technical risk exist.
  • Deploy appropriate software tools and skills for the design and implementation of an agent-based system.
  • Develop professional teamwork skills in a virtual learning environment.

3. Artefacts

6. Reflection: What? So What? What Next?

What?

This module introduced me to the idea of autonomous software agents that can sense, decide, act, and sometimes collaborate with other agents.

So What?

I realised that agent-based systems can be useful where problems involve distributed decision-making, dynamic environments, or automation. This connects strongly with my interest in AI-driven network operations.

What Next?

I plan to explore practical agent-based systems and understand how intelligent agents can support automation, troubleshooting, and decision support in enterprise infrastructure.
